Explain as Korean.

Account Pro는 보다 빠르게 사용자의 중요한 계정 정보를 관리 할 수 있습니다. 앱 내에 대시보드가 제공되어 빠르게 원하는 모바일 웹으로 접속 할 수 있으며, 기본 사파리 유저들은 Account Pro와 연동하여 쉽게 웹 로그인이 가능합니다.

Account Pro는 아래와 같이 제공합니다.

  1. 앱 내에 대시보드 아이폰 메인 화면과 같이 아이콘을 생성,표시하여 사용자가 쉽게 원하는 웹으로 접속이 가능합니다.
  2. 사파리와 연동 주요 사파리를 이용자들은 Account Pro앱을 통하여 쉽게 로그인이 가능하며, 등록, 검색이 용이합니다.
  3. 백업 서비스 지원(Dropbox, Wifi) 클라우드 서비스 Dropbox 이용자들은 Dropbox 백업 기능을 활성화하여, 데이터의 변화에 따라 자동적으로 백업 기능을 제공합니다. Wifi 백업은 같은 네트워크 내에서 보다 안전하게 백업이 가능합니다.
  4. 암호화 AES256 암호화 방식을 이용하여 중요 데이터 (계정, 비밀번호)를 안전하게 관리 할 수 있습니다.
  5. 태그 기능 계정 정보 중 태그를 입력함으로 써 보다 빠르고 쉽게 원하는 데이터를 검색 할 수 있습니다.
  6. 비밀번호 생성기 숫자, 문자, 특수 문자 조합을 이용하여 복잡한 비밀번호를 생성해줍니다.
  7. 보안 키보드 키보드 배열을 섞어 비밀번호 노출을 줄여줍니다.
