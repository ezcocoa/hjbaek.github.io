using mac os x
clojure에서는 종료 키 'Ctl+D'를 제공한다. 하지만 본좌는 emacs툴을 이용하기 때문에 종료키에 대한 키바인딩이 되어 있기 때문에 사용 할 수 없다. 그래서 아래와 같이 함수로 정의하여 사용 할 수 잇다.
[sourcecode langauge="clojure"]
(defn exit
"Returns to the OS by forcibly exiting the platform"
([] (. System exit 0))
([n] (. System exit n)))
[/sourcecode]
