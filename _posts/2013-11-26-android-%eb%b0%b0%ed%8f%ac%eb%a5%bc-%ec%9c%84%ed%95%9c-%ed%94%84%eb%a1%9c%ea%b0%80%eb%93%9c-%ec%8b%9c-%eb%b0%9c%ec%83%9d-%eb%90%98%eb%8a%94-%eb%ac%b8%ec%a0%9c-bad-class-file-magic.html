많은 안드로이드 개발자들이 앱 배포 시 프로가드를 적용하지 않는다. 이는 매우 위험하며, 물론 프로가드를 적용 시킨다고 해서 안전하지는 않다.

하지만 어느 정도 코드 유출과 보안 보장한다.

[sourcecode language=""] trouble processing: bad class file magic (cafebabe) or version (0033.0000) ...while parsing something.class$4.class ...while processing something$4.class [/sourcecode]

한 컴퓨터로 여러 가지 프로젝트를 진행 하다보니 나도 모르는 사이에 다른 프로젝트에서 문제가 일어나곤 한다.

입사 2개월이 넘었지만 개발에는 내 모든 장비를 사용하고 있으며, (물론 회사에 요청드리지 않았지만..)

그건 그렇고,

위 문제는 간단하게 SDK7.0에서 발생되는 문제로 SDK 6를 사용해주면 깔끔하게 문제가 해결된다.

사실 SDK 7을 사용하며, 다른 에러들을 다 제거하고 마지막으로 이 문제를 발견하였다.

먼저 이 문제를 발견했다면 이전에 발생된 문제는 무시해도 되는 문제였던 것이다.

어쨋든 간단하게 해결할 수 있었다.
