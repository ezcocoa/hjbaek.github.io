설치 버전 : cscope-15.8a
cscope 다운로드 : http://cscope.sourceforge.net/
아래와 같이 진행합니다.
[code]
$ ./configure
$ cd src/
$ emacs constants.h
[/code]
constants.h에 보면 TERMINFO값으로 BSD냐 아니냐 로 분기되어있습니다. 이를 아래와 같이 정의해줍니다.
[code]
#define TERMINFO 1
[/code]
이후 아래와 같이 마저 설치를 완료합니다.
[code]
$ cd ..
$ make
$ sudo make install
[/code]
이상입니다.
