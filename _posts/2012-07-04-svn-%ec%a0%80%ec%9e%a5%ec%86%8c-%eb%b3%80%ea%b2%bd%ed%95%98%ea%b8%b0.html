저장소 위치<디렉토리 또는 IP주소> 가 변경되었을 때 이를 적용하는 방법을 설명합니다.
Project 위치에서 아래와 같은 명령을 입력하면 현재 SVN의 상태를 확인 할 수 있습니다.
[code]
# svn info
[/code]
출력
[code]
Path: .
URL: https://xxx.xxx.xx.xxx/svn/Sample
Repository Root: https://192.168.0.1/svn/Sample
Repository UUID: 0xa34x90-971f-fb44-b96c-8xd5b45ea5b0
Revision: 733
Node Kind: directory
Schedule: normal
Last Changed Author: ezcocoa
Last Changed Rev: 717
Last Changed Date: 2012-05-29 10:57:15 +0900 (화, 29 5 2012)
[/code]
위와 같이 192.168.0.1로 설정되어있던 IP주소를 192.168.0.2로 변경한다면 아래와 같이 입력하면 된다.
[code]
# svn switch --relocate http://192.168.0.2/svn/Sample http://192.168.0.2/svn/Sample
[/code]
아무 에러가 발생되지 않는다면 정확히 입력된것이다.
