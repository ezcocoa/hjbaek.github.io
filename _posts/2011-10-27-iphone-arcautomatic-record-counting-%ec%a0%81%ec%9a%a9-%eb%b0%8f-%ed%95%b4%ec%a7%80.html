이번에 새로 나온 ARC 기능 활성화에 대해 일부분 소스코드에 비활성화 모드를 적용 시킬 수 있습니다.
문제는 이렇습니다.
기존 소스(ARC가 적용되어 있지 않은 소스 = release 메서드가 추가되어있는 소스)를 release를 제거 해줘야 하는데
이는 너무 많은 시간을 낭비하게 되죠.
실제 컴파일 단에서 release가 추가되는 거기때문에 때문에 보장만 한다면 작업할 필요가 없는 부분입니다.
이 문제를 해결 하기 위해서는
Project Info > Targets > Build Phases > Compiler sources 부분에서 Compiler Flags 단에 아래와 같은
Flag를 적용시켜주면 됩니다.
[code]-fno-objc-arc[/code]
이상입니다.
