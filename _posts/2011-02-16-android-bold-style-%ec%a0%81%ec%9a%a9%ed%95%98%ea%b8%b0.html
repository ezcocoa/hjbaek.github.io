TextView Bold 처리하기 (2가지방법)
1. html.fromHtml을 이용한 방법
- Html 태그를 이용한 방법이다. (bold, Underline, ltalic)
[sourcecode language="JAVA"]
textView.setText(Html.fromHtml(""+text+""));
[/sourcecode]
2. 그래픽을 이용한 방법
XML file
[sourcecode language="xml"]
android:textStyle="bold"
[/sourcecode]
[sourcecode language="java"]
textEdit.setPaintFlags(textEdit.getPaintFlags() | Paint.FAKE_BOLD_TEXT_FLAG);
[/sourcecode]
