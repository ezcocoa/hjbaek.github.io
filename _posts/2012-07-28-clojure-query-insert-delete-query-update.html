테이블명은 'users' 이며 두가지의 컬럼(fname, age)을 가지고 있습니다.

1. 데이터 추가하기

[sourcecode language="clojure"]
(insert-values
:users
["fname", "age"]
["hojun", "27"])
[/sourcecode]

2. 데이터 수정하기

[sourcecode language="clojure"]
(update-values
:users
["fname=?"
"hojun"]
{:age "28"})
[/sourcecode]

3. 데이터 검색하기

[sourcecode language="clojure"]
(with-query-results
results
["select fname from users"]
(doall results))
[/sourcecode]

4. 데이터 삭제하기

[sourcecode language="clojure"]
(delete-rows
:users
["fname=?ezcocoa"])
[/sourcecode]
