답은 간단하다.
0을 기준으로 그보다 높으면 양수 낮으면 음수로 판단하면된다.
아래와 같이 볼 수 있다.
if (timeInterval < 0 ) {
// this is positive value.
} else {
// this is negative value.
}

ABS함수를 통해 음수와 양수로 변환 할 수 있다.
아래와 같이 볼 수 있다.

int ToPositive(int value)
Return Abs(value)
int ToNegative(int value)
Return Abs(value) * -1
