static libraryies에서 기본적으로는 카테고리가 활성화되지 않습니다. 그래서 카테고리를 활성화 시켜주기위해서는
target > build settings > other linker flag 라는 속성 값에 두가지 항목을 넣어주어야 합니다.
[code]
-Objc
-all_load
[/code]
