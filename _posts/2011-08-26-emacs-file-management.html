 Key  Purpose
 Enter  Open the file
 q  Close the dir
 C  Copy file
 R  Rename file
 D  Delete file
 +  create a new dir
 Z  compress/decompress the file by gzip
 Key  Purpose
 m  mark a file
 u  unmark
 U  unmark all marked
 % m  mark by regular expression
 Key  Purpose
 g  refresh dir listing
 ^  go to parent dir
