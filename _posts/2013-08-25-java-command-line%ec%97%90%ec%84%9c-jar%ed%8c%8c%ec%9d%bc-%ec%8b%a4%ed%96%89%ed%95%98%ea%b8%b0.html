JAR파일은 JVM 위에서 돌릴 수 있다.

eg

foo.class 파일을 가지고 있다고 가정하자. (물론 static main 메서드도…) 

이 파일은 jar file이고. bar.jar라는 파일에 내장되어있다.

static main메서드를 사용하여 아래와 같이 실행시킬 수 있다.

# java -cp bar.jar foo

위에 클래스와 네임스페이스는 정확해야 한다. 그리고  ":" 콜론을 이용하여 추가적인 패스를 등록할 수 있다.
