iPhone Application Over the Air 배포 방식을 서술합니다.
이 방법은 다수의 사용자에게 배포 할때 자주 사용되며, 서버와 ADC ID Agent가 필요합니다.
이때 Identifier는 명확해야 합니다.
배포자(Server) <----> iPad User(Client)
아래와 같이 설정은 XCode 4기반으로 서술되었습니다.
1. Ad Hoc Provisioning Profile 다운로드합니다.
2. Project 선택 -> TARGET -> Build Settings -> Code Signing 항목 중 Distribution 항목을 설치 된 Ad Hoc 프로비저닝 프로파일 선택
3. Menu Product -> Archive 선택 (Archiving이 완료되면 Organizer 창이 뜸)
4. Share.. 버튼 클릭
5. Contents = iOS App Store Package (.ipa)로 선택
iDentity = 설치 된 Ad Hoc 프로비저닝 프로파일
6. Save for Enterprise Distribution을 선택하여 필수 항목을 적어줍니다.
Application URL (required) : http://

/app.ipa
Title (Required) : app
7. 저장 된 .ipa 파일과 .plist파일을 서버 경로에 위치시켜 줍니다.
8. 아래의 내용을 app.html로 저장하여 서버 경로에 위치시켜 줍니다.
[sourcecode language="html"]
Application 설치

Provisioning Profile 설치

[/sourcecode]
8. 이제 모바일 사파리를 통하여 접속하여 다운을 받을 수 있습니다.
