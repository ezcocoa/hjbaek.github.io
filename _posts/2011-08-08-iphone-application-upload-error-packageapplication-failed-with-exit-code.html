4.1 업데이트 이후 Application Upload 시 'PackageApplication failed with exit code'오류가 발생되었다면, 클래스 파일이 중첩되지 않았나 의심을 해보셔야합니다.
저같은 경우 작업 도중 SecurityViewController파일명이 Security?ViewController으로,
파일 내용에서는 Security_ViewController로 변경되어 긴 오류창과 함께 업로드가 중지되었습니다.
오류 로그를 복사하여 살펴보시면 Codesign check fails을 확인할 수 있습니다.
